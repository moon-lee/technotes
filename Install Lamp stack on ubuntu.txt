1. Download the latest Lamp stack from https://bitnami.com/stack/lamp

2. Support page 
	-> https://docs.bitnami.com/installer/faq/linux-faq/#what-is-a-bitnami-native-installer

3. Commnadline 
	-> sudo chown username:group <directory>
	-> sudo chown ownership:group <directroy>

4. Run Mysql
	-> sudo mysql --defaults-file=/opt/lampstack-5.6.28-1/mysql/my.cnf


5. Run stack
	-> Add alias to the shell with "alias"
		alias stack_start='sudo /opt/lampstack-5.6.28-1/ctlscript.sh start'
		alias stack_restart='sudo /opt/lampstack-5.6.28-1/ctlscript.sh restart'
		alias stack_stop='sudo /opt/lampstack-5.6.28-1/ctlscript.sh stop'
